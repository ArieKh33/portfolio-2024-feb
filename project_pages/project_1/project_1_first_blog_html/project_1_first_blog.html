Blog page - blog.html

What it does

When you press on the "Blog" link in the header you will land on this page with multiple blogs with different stories each, here I learnt how to link multiple websites to each other.

What I learnt

Since this website has multiple pages I thought to keep the styling compact with only using one folder, but after realizing how much harder it would be to track which styling belongs to which place I have made it a habit to make separate styling folders for each separate styling function. For example having a separate folder for the layout of a website and having a separate folder for the appearances such as the color fonts.
