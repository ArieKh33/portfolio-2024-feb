Main page - galgje.php

Starting out

To play the hangman game you need a word to start with, and you have 2 options to play with. The first being a random word from an array when you press the "willekeurig woord" button, and the second being an text input where you could write any word you'd like.

Important

The application is made in Dutch, however that does not mean that you are limited by only using Dutch words. The only limitations you do have are that you cannot use Capital letters and numbers.

Game start - spel.php

Playing the game

After you've picked a random word or a word chosen yourself you can start playing, the letter amount will be shown first and underneath will be the hangman itself. To play the game and guess the letters there is a line of buttons from the entire alphabet, and you can start whenever you'd like.

How to play the hangman game

For the people who don't know how the hangman game works I will give a quick rundown. In the hangman game you receive a random word that you will have to guess, the only hint you'll receive at the start is the amount of characters the word consists of. And you start by guessing which letters are a part of that word, but be careful because every time you guess a letter not part of the word a portion of the drawing of the "hangman" will be drawn out, and when the drawing is finished you lose, however if you guess a letter correctly then it'll show the letter in the right position of the word and when you guess all the letters correctly you'll be victorious.

Incorrect input - spel.php

What now?

So you have started playing and you start guessing that the letter "A" is part of the word, since the letter hasn't filled any blank spot and having the hangman drawn out more shows that that letter is Incorrect.

Can I continue?

Yes you can, since the drawing isn't finished yet, you could still try to guess with any other letter that isn't "A" and making the same mistake is thankfully not possible since you can't select a letter twice.

Correct input - spel.php

The right letter

When you've guessed a letter that is part of the word it will show up in the position it belongs

Multiple of the same letter

Let's say that you have a word with 2 "E" in there, and you're worried that you can't guess the letter twice for each letters since the button disables after use, but there is no need to worry since duplicate letters will be filled in automatically.

Winner - spel.php

You've won

After filling out all the blanks You will receive a win screen with the word fully spelled out.

Where do I go now?

in this page is a button in the bottom left corner that will send you back to the home screen where you could play again.

Loser - spel.php

You've lost

When the drawing of the hangman is finished you will lose and get to a losing screen where it will display the correct word you were trying to guess.

Can I keep playing?

No matter how many times you lose you'll always be able to play another game with a random word or a word chosen yourself, practise makes perfect.
