Database website

When I started learning PHP and SQL and found out I could use them together, I started to look around to see how I could use them both in 1 application, and that is how I came up with making a website that has a databases of movies and shows, and it has the added option of changing/adding values to the database as well.

Main page New Media Movie data Shows data Edit page

Main page - Index.php

The start

Since this website has the option to add/change values to the database and to view them I have made a menu to pick what you want to do with the databases.

New Media - insert.php

Adding values

Since this website has information about movies and shows we need to have it clear which is which. in this page you have textarea's where you could insert any kind of movie/show from any time and if you save it by selecting movie or show under "type media" and pressing "opslaan" to submit it.

Where is the data?

The data is stored in 2 databases using the POST GET method in PHP to send the data in the database.

Movie data - detail.php

Displayed data

For each list of data there is a column which will show exact data from the database.

Accessing data

If some details of a movie/show are incorrect there is always a way to change them instead of having to delete everything and start over.

Shows data - detail.php

Another table

To make the overview less confusing I've made another table which contains all the data for the shows in the database.

Edit page

Let's say that you accidentally misspelled a name of a movie and that you'd want to change it. In the blue text named "bekijk details" is where you get directed to the edit page.

Edit page - edit.php

Changing content

After pressing on a movie/show where you'd like to change some values on you come to this page where you can literally change everything,

Extensive change

For an example let's give an extreme scenario where the only information being correct would be the title and the type of media it is, well here you can change every bit of that and even change it to a different form of media if there needs to be correction there too.
